# BLACKWIND.TMN — статичный лендинг

Готовый адаптивный сайт для детейлинг-студии.

## Что внутри
- `index.html` — главная страница
- `assets/css/style.css` — стили
- `assets/js/main.js` — анимации, галерея, сравнение до/после
- `assets/images` — оптимизированные изображения WebP/JPG
- `assets/video` — сжатое видео для акционного блока

## Что заменить перед публикацией
1. Телефон в ссылках `tel:+7XXXXXXXXXX` и `wa.me/7XXXXXXXXXX`.
2. Точный адрес, режим работы и ссылки на мессенджеры.
3. `https://example.com/` в `sitemap.xml` на реальный домен.
4. При подключении формы — добавить backend, CRM или сервис заявок.

Откройте `index.html` в браузере или загрузите папку на хостинг.
